# API 종류
## App
- 환자 로그인 
- 환자 회원가입 
- 환자 로그아웃 
- 의료진 앱 진료실 QR스캐너 -> 진료 접수
- 삼변측량에 필요한 전체 비콘 리스트 반환
- 진료 동선 안내 (다익스트라 알고리즘을 통한 최단 경로 노드)
- 동선 목록에서 해당 동선을 선택 시 최단 경로 반환
- 검색을 통한 내비게이션에 필요한 방 정보 리스트 반환 
- 검색을 통한 내비게이션의 최단 경로 반환
- 검색을 통한 내비게이션에서 현재 위치를 통한 최단 경로 반환
- 대기순번 (pusher를 사용한 실시간 대기순번)
- 미결제 진료비 내역
- 결제된 진료비 내역 조회
- 이전 진료 동선 조회

## 의료진 Web
- 의료진 로그인
- 의료진 회원가입 (정해진 의료진만 회원가입 가능)
- 의료진 로그아웃
- 환자 데이터 생성 (앱 회원자인지 판별 후 create, update) 
- 환자 이름으로 검색 (동명이인 목록 반환)
- 환자 목록에서 선택 후 환자 정보 조회(환자 정보, 최근 진료 데이터, 진료 동선) 
- 환자 진료 접수 및 업데이트 (QR코드 접수로 알 수 없는 데이터 : 의사 이름, 진료 시간, 진료비 등) 
- 진료 정보 가져오기 (진료과, 진료실, 의사)
- 해당 환자 진료 종료 버튼 클릭 시 대기순번에 반영
- 환자 현재 날짜 진료 데이터 가져오기 
- 동선 설정에 필요한 방 정보 가져오기 
- 환자 진료 동선 설정 

## 관리자 Web
- 관리자 로그인
- 관리자 로그아웃
- 비콘 메인페이지 - 비콘 위치 및 정보 가져오기
- 비콘 정보 수정
- 비콘 정보 검색 
- 노드 메인페이지 - 노드 위치정보 가져오기
- 노드 정보 수정
- 노드 거리 메인 페이지 - 노드 간 설정 정보 및 거리 값 가져오기
- 병원 방과 노드 연결하기 
- 방 정보 업데이트
